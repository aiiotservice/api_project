# .NET Core Performance & CI/CD Project

## 📌 Project Summary
This project is developed using **.NET Core** with a primary focus on performance evaluation and optimization.  
We apply various performance metrics, analyze trade-offs, and capture detailed insights into **CPU usage** and **memory consumption**.  

In addition, we design and implement a **CI/CD pipeline** to automate build, test, and deployment processes, and share the configuration details for reproducibility.

---

## 🚀 Key Highlights
- Application built on **.NET Core**
- Performance monitoring:
  - CPU usage tracking
  - Memory usage analysis
  - Trade-off evaluation
- CI/CD pipeline creation and documentation

---

## 🎯 Goals
- Ensure efficient resource utilization  
- Identify and address performance bottlenecks  
- Automate deployment workflows for faster delivery  

---

## 📈 Deliverables
- Performance reports (CPU & memory usage)  
- Trade-off analysis documentation  
- CI/CD pipeline setup details
